# Installation
Add jitpack to your project `build.gradle` 
```groovy
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
```
and then add **mvi-observable** to the list of dependencies
```groovy
dependencies {
    implementation 'com.github.User:Repo:Version'
}
```
